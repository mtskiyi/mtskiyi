* **Window Management:** Hyprland managed exclusively through native Lua configurations

* **Design Pattern:** Keyboard-driven workflows, strict minimalism, tiling layouts

### Software Development
* **Languages:** Python
* **TUI Engineering:** Building text-based user interfaces via Rich and Textual frameworks
* **Automation:** Custom CLI tooling, JSON report validation scripts, development environment orchestration

### Hardware Modification
* **Platform Optimization:** AMD AM4 hardware configurations, custom BIOS flashing pipelines
* **Mobile Diagnostics:** Xiaomi hardware modification, bootloader unlocking, custom ROM deployment (PixelOS), structural clear modifications

## Core Projects

### hyprland-lua
Modular configuration framework translating complete environment management rules directly into functional Lua code for performance optimizations.

### opencore-validator
A automated Python parser designed to ingest, validate, and check `hardware_report.json` parameters against strict OpenCore bootstrap rules to ensure boot compatibility.

### python-tui-tools
A collection of custom lightweight system tools implementing the Textual library to manage operations via low-overhead terminal interfaces.
